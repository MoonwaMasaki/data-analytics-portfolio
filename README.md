# Data Analytics Portfolio
----

### 👋 Hey there

Welcome to my data playground, where raw datasets get cleaned up, questioned, visualized, and turned into stories that actually make sense.

This repo is a growing collection of data analytics projects where I explore patterns, test ideas, and turn numbers into insights that (hopefully) don’t put you to sleep.

Think of it as:

📉 chaos in data → 🧹 cleaning → 📊 visuals → 💡 “ohhh that’s what’s going on”

----


### 🎯 Purpose (What it's about)

This isn’t just about writing Python code.

It’s about thinking in questions before answers.

Each dataset is an investigation:

- What’s actually happening here—and what’s being hidden?
- Where do patterns break, and why?
- What would a decision-maker actually do with this?

This work focuses on turning raw data into:

- insights stakeholders can act on
- dashboards that simplify complexity
- stories that survive beyond the chart itself

Every project here is part analysis, part storytelling, and part “wait… that’s actually interesting.”
